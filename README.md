# Sitio_Web_con_Bootstrap

Landing page de portfolio personal desarrollada con Bootstrap 5 para el taller practico de creacion y despliegue de un sitio web responsivo.

## Contenido del proyecto

- Navbar responsiva fija en la parte superior
- Carousel principal con 3 imagenes
- Seccion de proyectos con cards y botones
- Modal dinamico con informacion adicional de cada proyecto
- Formulario de contacto con estilo Bootstrap
- Recursos listos para publicar en GitHub Pages
